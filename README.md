# kindness-module-manage

#### 介绍
Webman 模块管理插件

#### 安装

composer require kindness/module-manage


#### 模块创建

执行命令：php webman module:create

#### 模块列表

执行命令：php webman module:list

#### 模块禁用

执行命令：php webman module:stop <模块名称>

#### 模块启用

执行命令：php webman module:start <模块名称>
